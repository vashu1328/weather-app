# weather-app
Weather App built using HTML, CSS, and JavaScript that shows real-time weather data using API with a clean and responsive UI.
A simple Weather App built using HTML, CSS, and JavaScript that shows real-time weather information using Weather API.

## Features

- Search weather by city name
- Shows temperature, humidity, and weather condition
- Real-time API data
- Clean and responsive UI
- Beginner friendly project

## Tech Stack

- HTML
- CSS
- JavaScript
- Weather API


## ▶️ How to Run

1. Clone the repository
2. Open index.html in browser
3. Enter city name
4. Get weather details
